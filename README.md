# myndbodi.com

The marketing website for **Chronos by Mynd & Bodi Institute (MBI)** — a daily wellness intelligence app for Apple Watch that produces one score, two drivers, one narrative line, and one nudge each morning, scored only against the user's own baseline. The site is built with Next.js 16 (App Router) and React 19, styled with Tailwind CSS v4 (design tokens live in `app/globals.css` under `@theme`), and uses a `ThemeProvider` world-switching pattern (`mbi` / `chronos` / `legal`). It deploys automatically to Vercel from the `main` branch.
